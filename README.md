# Midnight Boxing Championship

![](https://github.com/DQuanBui/pro_boxing_game/blob/main/boxing.avif)

## Project Overview

**Midnight Boxing Championship** is an interactive browser-based boxing game built with **HTML, CSS, and JavaScript**. The game challenges players to win rounds, score knockdowns, manage stamina, build combos, block incoming punches, dodge counters, and defeat an AI-controlled opponent in a professional boxing match.

The project focuses on real-time canvas rendering, responsive front-end gameplay, career progression, upgrade systems, reward feedback, and an engaging user experience directly in the browser.

## Tools

This project uses:

- **HTML5** for the page structure, menus, HUD, and canvas setup
- **CSS3** for responsive layout, visual styling, overlays, controls, and game interface design
- **JavaScript** for gameplay logic, AI behavior, career progression, and game systems
- **HTML5 Canvas API** for real-time ring, fighter, crowd, particle, and combat rendering
- **LocalStorage** for saving career progress, XP, cash, upgrades, and achievements
- **Git and GitHub** for version control and project hosting
- **GitHub Pages** for online deployment

## Gameplay

The player enters the ring as a blue-corner boxer and fights against an AI-controlled opponent from the red corner. The goal is to win by knockout or by judges decision after the selected number of rounds.

Players must manage stamina carefully because punches, blocking, and dodging all consume energy. Jabs are fast and safe, crosses deal stronger damage, hooks and uppercuts hit harder but recover slower, and power shots become available after building the power meter. Blocking reduces incoming damage, while dodging can avoid punches and create counter opportunities.

Career mode makes each fight more meaningful. Players earn XP, cash, and bonus points after fights, complete objectives, unlock achievements, and spend cash on upgrades that improve boxing performance over time.
